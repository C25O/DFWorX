# Tehcnologies & Tools

## **1. Code Management & Collaboration**

* **GitHub** — Platform for hosting, version control, and collaboration using Git repositories.
* **Claude Code** — AI coding assistant that helps generate, debug, and refactor code through natural language.

## **2. Frontend Development**

* **Next.js** — React-based full-stack framework for building optimized web applications with SSR/SSG capabilities.
* **React** — Declarative JavaScript library for building dynamic user interfaces.
* **Tailwind CSS** — Utility-first CSS framework for rapid UI development with consistent design.
* **Shadcn UI** — Prebuilt UI component library built on Radix and Tailwind CSS for React apps.

## **3. Backend & Database**

* **FastAPI** — Modern, high-performance Python framework for building APIs with automatic validation.
* **Supabase** — Open-source Firebase alternative offering Postgres, auth, and storage APIs.
* **ConvexDB** — Reactive backend platform that manages real-time data and business logic automatically.

## **4. Environment & Dependency Management**

* **PNPM / Yarn** — Efficient JavaScript package managers for handling project dependencies.
* **UV / VENV** — Python environment tools for isolating and managing dependencies across projects.

## **5. Deployment & DevOps**

* **Docker** — Containerization platform for packaging and deploying applications consistently across environments.
* **Coolify** — Self-hosted PaaS that simplifies deploying and managing apps without cloud vendor lock-in.
